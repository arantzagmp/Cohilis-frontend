Descripción del Proyecto

Este proyecto es una aplicación web front-end responsiva desarrollada para Cohilis, una empresa enfocada en servicios de recolección, transporte y manejo de residuos industriales. El objetivo principal del sitio es brindar una presencia digital profesional, mostrar los servicios de la empresa, certificaciones e información de contacto, además de ofrecer una experiencia intuitiva para los usuarios en computadora, tablet y celular.

La aplicación fue desarrollada con React y utiliza una arquitectura basada en componentes para mantener el código ordenado, escalable y fácil de mantener.

Funcionalidades Principales

- Diseño responsivo adaptable a diferentes tamaños de pantalla.
- Navegación entre secciones mediante React Router.
- Componentes reutilizables para una mejor estructura del proyecto.
- Formulario de contacto para solicitudes de clientes.
- Diseño moderno utilizando Flexbox y Grid.
- Integración con API externa para búsqueda de información.
- Preloader durante las solicitudes a la API.
- Renderizado dinámico de resultados en bloques de 3 elementos.
- Botón “Mostrar más” para cargar resultados adicionales.
- Mensaje cuando no se encuentran resultados.
